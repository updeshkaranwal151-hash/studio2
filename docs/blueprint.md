# **App Name**: CalcZen

## Core Features:

- Basic Arithmetic: Perform addition, subtraction, multiplication, and division operations.
- Scientific Calculations: Handle trigonometric, logarithmic, and exponential functions.
- Unit Conversions: Convert between length, mass, and temperature units.
- Calculation History: Maintain a detailed history of calculations with copy and reuse options.
- Currency Conversion: Convert between different currencies, automatically fetching latest values using a tool to decide whether to incorporate that value into its output.
- Memory Storage: Quick memory storage features (M+, M-, MR, MC).
- Graphing View: Visualize equations through a graphing interface.

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) for a professional yet creative feel.
- Background color: Dark gray (#282828) to ensure high contrast and readability in dark mode.
- Accent color: Vibrant orange (#FF4500) to highlight important functions and buttons.
- Body and headline font: 'Inter', a sans-serif font that provides a modern, neutral look, is appropriate for both headlines and body text.
- Use minimalistic icons for operations and functions to maintain a clean design.
- Employ a grid-based layout for easy tapping, especially on mobile devices.
- Implement subtle animations for button presses and result transitions to provide visual feedback.
